# Идея RAG-чата: Помощник по законодательству РФ с пояснениями «Российской газеты»

## 1. Проблема
Понимание юридических норм требует времени и знания контекста. При этом «Российская газета» публикует официальные комментарии (упрощённые версии), но они разрозненны и не интегрированы в единую поисковую систему. Пользователи не могут быстро найти одновременно оригинал документа и его простое разъяснение.

## 2. Цель проекта
Создать RAG-чат, который по запросу пользователя находит наиболее релевантный оригинальный документ и соответствующий комментарий, а затем формирует понятный ответ.

## 3. Для кого проект
- Граждане РФ, желающие разобраться в законах (налоги, права, COVID-ограничения и т.д.)
- Юристы и правоведы (для быстрого поиска связки «оригинал + комментарий»)
- Журналисты, пишущие о законодательстве

## 4. Какие данные используете
- **Источник:** Датасет с парами «оригинал + упрощённая версия (комментарий РГ)»
- **Объём:** 2 963 пары документов
- **Столбцы:** Название документа, ссылка, полный текст оригинала, комментарий РГ, дата публикации

## 5. Критерий успеха
- **Точность:** Для 90% тестовых вопросов система возвращает правильную пару (оригинал + комментарий) из датасета.
- **Полезность ответа:** Ответ RAG-чата содержит опирается на оригинальный документ, цитату из него и пересказ комментария РГ.
- **Скорость:** Ответ формируется не дольше 10 секунд.
- **Покрытие:** Чат умеет отвечать на вопросы о любом документе из датасета.
- **Пользовательский фидбек:** Не менее 80% тестовых пользователей оценивают ответ как «понятный и соответствующий источнику».
